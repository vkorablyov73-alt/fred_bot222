### Материал для отличников по теме «Логарифмы» для уровня 3 (звёздочка)

#### Введение

Логарифмы — это мощный инструмент, который находит применение в различных областях математики и естественных наук. В этом материале мы рассмотрим нестандартные подходы к решению задач с логарифмами, докажем теоремы двумя способами и предложим задачи для самостоятельного решения.

#### Теорема о свойствах логарифмов

**Теорема:** Пусть $a > 0, a \neq 1$, $b > 0$, $c > 0$. Тогда:

1. $\log_a(bc) = \log_a b + \log_a c$
2. $\log_a \left(\frac{b}{c}\right) = \log_a b - \log_a c$
3. $\log_a b^c = c \log_a b$

**Доказательство первого способа:**

1. Рассмотрим первое свойство. Пусть $x = \log_a b$, $y = \log_a c$. Тогда по определению логарифма $a^x = b$ и $a^y = c$.
2. Перемножим эти равенства: $a^{x+y} = bc$.
3. По определению логарифма $\log_a(bc) = x + y = \log_a b + \log_a c$.

**Доказательство второго способа:**

1. Рассмотрим первое свойство. Пусть $x = \log_a b$, $y = \log_a c$. Тогда по определению логарифма $a^x = b$ и $a^y = c$.
2. Разделим первое равенство на второе: $\frac{a^x}{a^y} = \frac{b}{c}$.
3. По свойству степеней $a^{x-y} = \frac{b}{c}$.
4. По определению логарифма $\log_a \left(\frac{b}{c}\right) = x - y = \log_a b - \log_a c$.

Аналогично можно доказать второе и третье свойства.

#### Нестандартные подходы к решению задач с логарифмами

Рассмотрим несколько нестандартных подходов к решению задач с логарифмами.

**Задача 1:** Решите уравнение $\log_2(x^2 - 4) = 3$.

**Решение:**

1. По определению логарифма $x^2 - 4 = 2^3$.
2. Решим полученное уравнение: $x^2 - 4 = 8$.
3. $x^2 = 12$.
4. $x = \pm \sqrt{12} = \pm 2\sqrt{3}$.

**Задача 2:** Решите неравенство $\log_3(x + 5) > 2$.

**Решение:**

1. По определению логарифма $x + 5 > 3^2$.
2. Решим полученное неравенство: $x + 5 > 9$.
3. $x > 4$.

**Задача 3:** Решите систему уравнений:

$\begin{cases}
\log_2 x + \log_2 y = 3 \\
x + y = 16
\end{cases}$

**Решение:**

1. Из первого уравнения по свойству логарифмов $\log_2(xy) = 3$.
2. По определению логарифма $xy = 2^3 = 8$.
3. Подставим $y = 16 - x$ во второе уравнение: $x(16 - x) = 8$.
4. Решим полученное уравнение: $x^2 - 16x + 8 = 0$.
5. $x = \frac{16 \pm \sqrt{256 - 32}}{2} = \frac{16 \pm \sqrt{224}}{2} = 8 \pm 2\sqrt{14}$.
6. $y = 16 - x = 8 \mp 2\sqrt{14}$.
